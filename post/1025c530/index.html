2.3变量值互换

在Go语言中两个变量互换的操作十分简单

1
2
3
4
5
6
7

var (                //定义初始化变量
	m    int = 1
	n    int = 2
	temp int = 6
)
m,n = n,m           //将m与n的变量值互换
fmt.Println(m, n)   //2 1

2.4_丢弃变量

1
2

//_丢弃变量任何赋予它的值都会被丢弃，该变量不占用命名空间
_,d := 1,2    //此时d的值为2，1被丢弃掉
